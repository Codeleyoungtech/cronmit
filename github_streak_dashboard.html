Cronmit

A GitHub streak keeper and 12-week contribution graph designer. It never uses empty commits: every run writes real audit data before committing, and art mode keeps a one-commit floor on every day.

Contribution graph painter

These commands are meant for your terminal, where GitHub CLI is already authenticated. The dashboard does not ask for a GitHub token.

          


          

            

              
1Use this dashboard to choose mode, commit count, timezone, and the 12-week art plan.

              
2Download or copy the generated files into .github/workflows/cronmit.yml, scripts/commit.sh, and cronmit-plan.json.

              
3Run the generated GitHub commands to set COMMITS_PER_DAY, CRONMIT_MODE, and Actions permissions.

              
4Push the repo, then run the workflow manually once from the Actions tab or with the generated gh workflow run command.

              
5If the repo is private, enable private contributions on your GitHub profile.

            

            
Cronmit does not bypass branch protection. If your default branch blocks GitHub Actions pushes, use a dedicated streak repo or loosen rules for the Actions bot.
